Get started with Bitcoin II

Before you start using Bitcoin II, there are a few things that you need to know in order to use it securely and avoid common pitfalls.

1. Inform Yourself

Bitcoin II is different than what you know and use every day. You should take some time to inform yourself before using BC2 for any serious transaction. Learn more

2. Choose your wallet

You can install a Bitcoin II wallet on your mobile device or computer. In just a few easy steps you will be ready to start making transactions. See all wallets

3. Get BC2

You can get BC2 by accepting it as a payment for goods and services or by buying it from a friend or someone near you. You can also buy it directly from an exchange. Find an exchange
